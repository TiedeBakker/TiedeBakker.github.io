Mijn Zakboek

Mijn Zakboek (MZB) zet ik op als een Progressive Web Application met als idee dat ik op iPhone en tablet allerlei informatie onder handbereik heb. Omdat ik het op dit moment via een publieke https-server (Github.com) publiceer zet ik er geen vertrouwelijke en privé informatie op; alleen informatie die eigenlijk ook via andere routes te vinden is.

De inhoud stem ik qua omvang af op gebruik via iPhone/tablet, maar ik wil ook extra informatie kunnen bieden die op laptop/desktop goed te presenteren. De applicatie moet (in ieder geval op termijn) responsive worden.

In deze (voorlopige) toelichting probeer i een aantal keuzes die ik bij het ontwerpen van MZB maak toe te lichten, zodat ik t.z.t. nog weet waarom ik bepaalde keuzes heb gemaakt en waarmee ik kan terugvinden hoe e.e.a. werkt.
