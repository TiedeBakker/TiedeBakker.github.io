Mijn Digitale ZakBoek

Deze App gebruik ik om informatie die ik altijd onder handbereik wil hebben te organiseren.

Editor

Voor het snel aanpassen van HTML-teksten gebruik ik een online WYSIWYG-editor (CKEditor), waarin ook allerlei extra code kan worden ingevoegd. Een probleem van de broncode is dat er erg veel witregels tussen komen. Deze kan ik eventueel wegwerken met een online-tool als code verbeteraar (hij heet Code Beautify.

organismen

Organismen module

In de organismen module wil ik het mogelijk maken door een lijst met organismen te bladeren. Ook moet ik de mogelijkheid hebben een filter te zetten. Het is prettig om op elk moment het zoekvenster met één klik te kunnen bereiken. Bij elk organisme probeer ik één of meer afbeeldingen te selecteren die in het veld handig zijn. Ik moet door die individuele afbeeldingen kunnen bladeren/swipen; eventueel via een klik vooruit/achteruit
